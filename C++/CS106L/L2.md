#### Overloading

Define two functions with the same name but different call signature

函数名相同，签名不同

```c++
double func(int x) {
  return (double)x + 3;	// typecast: int -> double
}

double func(double x) {
  return x * 3;
}
func(2) // 5.0
func(2.0) // 5.0
```
