## 在Linux中如何查看当前目录文件大小, 并从大到小排列?

```bash
du -h --max-depth=1 | sort -hr
```





## How to open a tar file

```bash
tar -xvf file.tar
```



##  What about tar.gz

```bash
tar -zxvf file.tar.gz
```
